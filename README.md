# Portfolio Website – Ayyoub

Welcome to my personal developer portfolio! This website showcases my **projects, skills, and experience** as a full-stack developer. It’s designed to highlight the work I’ve done in web development, cloud services, and mobile applications.

---

## Features

- **Projects Section:** Highlighting key projects.
- **Skills & Tech Stack:** Organized overview of my front-end, back-end, database, and cloud skills.  
- **Experience Summary:** Brief overview of professional experience and roles.  
- **Contact Section:** Easy ways to reach me, including email, LinkedIn, and GitHub.  
- **Resume Download:** Direct link to a PDF version of my resume.

---

## Usage

This is a static portfolio website, but you can clone the repository to run locally:

```bash
git clone https://github.com/ayyoub512/ayyoub.io.git
cd ayyoub.io
npm install
npm run dev
